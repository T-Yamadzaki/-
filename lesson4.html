Урок 4: Условные операторы

Введение

Преподаватель: На предыдущем уроке мы изучали переменные и типы данных. Сегодня мы поговорим об условных операторах. Условные операторы позволяют выполнять различные действия в зависимости от условий. Это основа для создания логики в вашем коде.

В конце урока вы научитесь использовать условные операторы if, else и switch.

Теоретическая часть

Преподаватель: Условные операторы проверяют, выполняется ли определённое условие, и в зависимости от этого выполняют определённый код. Давайте рассмотрим основные виды условных операторов в JavaScript:

  • if — выполняет код, если условие истинно.
  • else — выполняет код, если предыдущее условие ложно.
  • else if — проверяет дополнительное условие, если предыдущее условие ложно.
  • switch — выполняет код в зависимости от значения выражения.

Важно отметить, что в JavaScript, в отличие от C++, условные операторы могут проверять не только равенство, но и неравенство, больше или меньше. Также в Python используется аналогичная логика, но с некоторыми синтаксическими отличиями. Например, в Python отступы играют важную роль в определении блоков кода.

Пример кода

Преподаватель: Давайте посмотрим на простой пример, где мы проверим возраст пользователя и выведем сообщение в зависимости от этого:

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Условные операторы в JavaScript</title>
</head>
<body>
    <script>
        let возраст = 18;

        if (возраст < 18) {
            console.log("Вы ещё слишком молоды для этого.");
        } else if (возраст >= 18 && возраст < 65) {
            console.log("Добро пожаловать!");
        } else {
            console.log("Вы заслуживаете уважения!");
        }
    </script>
</body>
</html>

Преподаватель: В этом коде мы проверяем возраст пользователя и выводим соответствующее сообщение. Обратите внимание на оператор &&, который проверяет несколько условий одновременно.

Домашнее задание

1. Создайте переменную для хранения числа от 1 до 7 и используйте оператор switch для вывода названия дня недели в зависимости от числа.

let день = 3;

switch (день) {
    case 1:
        console.log("Понедельник");
        break;
    case 2:
        console.log("Вторник");
        break;
    case 3:
        console.log("Среда");
        break;
    case 4:
        console.log("Четверг");
        break;
    case 5:
        console.log("Пятница");
        break;
    case 6:
        console.log("Суббота");
        break;
    case 7:
        console.log("Воскресенье");
        break;
    default:
        console.log("Неизвестный день");
}

2. Попробуйте изменить код так, чтобы вместо названия дня недели выводилось, является ли день рабочим или выходным.

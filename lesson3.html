Урок 3: Циклы в JavaScript

Введение

Преподаватель: На этом уроке мы изучим, что такое циклы в JavaScript и как они помогают повторять выполнение кода.

Что такое цикл?

Цикл — это конструкция, которая позволяет выполнять блок кода несколько раз. В JavaScript существует несколько типов циклов, таких как for, while и do...while.

Цикл for

Преподаватель: Цикл for используется, когда мы знаем заранее, сколько раз нужно выполнить блок кода. Его структура следующая:

for (инициализация; условие; инкремент) {
    // код, который будет выполняться
}

Теперь давайте разберем каждый элемент:

  • Инициализация: Это начальное значение счетчика, например, let i = 0. Эта часть выполняется только один раз перед началом цикла.
  • Условие: Это логическое выражение, которое проверяется перед каждой итерацией. Если условие истинно, цикл продолжает выполнение. Например, i < 5.
  • Инкремент: Это выражение, которое выполняется после каждой итерации. Обычно оно увеличивает (или уменьшает) значение счетчика, например, i++.

Вот пример цикла for, который выполняется 5 раз и выводит значения переменной i:

for (let i = 0; i < 5; i++) {
    console.log(i);
}

Цикл while

Преподаватель: Цикл while выполняет блок кода, пока заданное условие истинно. Если условие никогда не становится ложным, цикл может выполняться бесконечно.

let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

В этом примере цикл будет выполняться до тех пор, пока i меньше 5.

Цикл do...while

Преподаватель: Цикл do...while выполняет блок кода как минимум один раз, а затем проверяет условие. Если условие истинно, цикл повторяется.

let i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);

В этом примере блок кода будет выполнен как минимум один раз, даже если i изначально равен 5.

Интересные факты о циклах

Важно понимать, что циклы в JavaScript, как и в других языках, могут вызывать ошибки, если не контролировать условия завершения. Например, бесконечные циклы могут привести к зависанию страницы.

В отличие от статически типизированных языков, таких как C++, JavaScript позволяет гибко изменять тип переменной в цикле, что может привести к неожиданным результатам, если не следить за типами данных.

Теоретическая часть

Преподаватель: Циклы являются мощным инструментом, позволяющим автоматизировать задачи и обрабатывать массивы данных. Например, вы можете использовать цикл для перебора элементов массива и выполнения операций с каждым элементом.

Практическая часть

Задание: Напишите код, который будет выполнять следующее:

  • Используйте цикл for, чтобы вывести все четные числа от 0 до 10.
  • Используйте цикл while, чтобы вывести числа от 5 до 0.
  • Используйте цикл do...while, чтобы спросить пользователя, хочет ли он продолжить, и продолжайте, пока он отвечает "да".

Пример решения:

// Четные числа от 0 до 10
for (let i = 0; i <= 10; i += 2) {
    console.log(i);
}

// Числа от 5 до 0
let j = 5;
while (j >= 0) {
    console.log(j);
    j--;
}

// Запрос у пользователя
let answer;
do {
    answer = prompt('Хотите продолжить? (да/нет)');
} while (answer === 'да');

Попробуйте реализовать это задание самостоятельно и запустите ваш код в браузере!

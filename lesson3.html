Урок 3: Циклы

Введение

Преподаватель: На прошлом уроке мы изучали условные операторы. Сегодня мы поговорим о циклах. Циклы позволяют повторять выполнение одного и того же блока кода несколько раз, что делает код более эффективным.

В конце урока вы сможете использовать циклы for, while и do...while.

Теоретическая часть

Преподаватель: Циклы в JavaScript позволяют выполнять повторяющиеся операции. Основные типы циклов:

  • for — используется, когда известное количество итераций.
  • while — выполняет цикл, пока условие истинно.
  • do...while — сначала выполняет код, а затем проверяет условие.

Интересный момент: в JavaScript, в отличие от C++, переменные в циклах не обязательно должны быть статично типизированными. Например, в C++ мы объявляем тип переменной заранее, тогда как в JavaScript тип может изменяться динамически. В Python также используются аналогичные циклы, но синтаксис может отличаться.

Пример кода

Преподаватель: Давайте рассмотрим пример использования цикла for для вывода чисел от 1 до 5:

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Циклы в JavaScript</title>
</head>
<body>
    <script>
        for (let i = 1; i <= 5; i++) {
            console.log(i);
        }
    </script>
</body>
</html>

В этом коде цикл for будет повторяться, пока i меньше или равно 5, и на каждой итерации будет выводиться текущее значение i.

Домашнее задание

1. Напишите цикл while, который будет выводить числа от 1 до 10.

let j = 1;

while (j <= 10) {
    console.log(j);
    j++;
}

2. Используйте цикл for для вывода квадратов чисел от 1 до 5.

for (let k = 1; k <= 5; k++) {
    console.log(k * k);
}

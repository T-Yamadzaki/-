Урок 6: Объекты и массивы

Введение

Преподаватель: На прошлом уроке мы изучали функции. Сегодня мы поговорим об объектах и массивах. Эти структуры данных позволяют хранить и организовывать данные в JavaScript, что является основой для работы с более сложными приложениями.

В конце урока вы сможете создавать и манипулировать объектами и массивами.

Теоретическая часть

Преподаватель: Объекты в JavaScript — это коллекции пар "ключ-значение". Они позволяют организовывать данные в более сложные структуры. Массивы, с другой стороны, — это упорядоченные коллекции значений. Основные моменты:

  • Объекты создаются с помощью фигурных скобок {}.
  • Массивы создаются с помощью квадратных скобок [].
  • Элементы массива имеют числовые индексы, а свойства объекта — строковые ключи.

Интересный момент: в отличие от C++, где массивы имеют фиксированный размер, в JavaScript массивы могут динамически изменяться. Python также имеет динамические списки, что делает работу с данными более гибкой.

Пример кода

Преподаватель: Давайте рассмотрим пример создания объекта и массива:

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Объекты и массивы в JavaScript</title>
</head>
<body>
    <script>
        // Создание объекта
        let человек = {
            имя: 'Иван',
            возраст: 25,
            профессия: 'разработчик'
        };

        console.log(человек.имя); // Выводит "Иван"

        // Создание массива
        let числа = [1, 2, 3, 4, 5];
        console.log(числа[0]); // Выводит 1
    </script>
</body>
</html>

В этом коде мы создаём объект человек с тремя свойствами и массив числа с пятью элементами.

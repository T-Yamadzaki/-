Урок 7: Условия и логические операторы

Введение

Преподаватель: На прошлом уроке мы изучали объекты и массивы. Сегодня мы поговорим об условиях и логических операторах. Условия позволяют принимать решения в коде, а логические операторы помогают комбинировать эти условия.

В конце урока вы сможете использовать конструкции if, else и switch.

Теоретическая часть

Преподаватель: Условия в JavaScript позволяют выполнять разные действия в зависимости от истинности выражений. Основные конструкции:

  • Конструкция if проверяет условие и выполняет код, если условие истинно.
  • Конструкция else выполняется, если предыдущее условие ложно.
  • Конструкция switch используется для выбора одного из нескольких блоков кода для выполнения.

Интересный момент: в JavaScript логические операторы && (и), || (или) и ! (не) могут использоваться для комбинирования условий. Это аналогично C++ и Python, но синтаксис может немного отличаться.

Пример кода

Преподаватель: Давайте рассмотрим простой пример использования условий:

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Условия в JavaScript</title>
</head>
<body>
    <script>
        let возраст = 18;

        if (возраст >= 18) {
            console.log('Вы совершеннолетний.');
        } else {
            console.log('Вы несовершеннолетний.');
        }
    </script>
</body>
</html>

В этом коде мы проверяем, является ли человек совершеннолетним, и выводим соответствующее сообщение.

Домашнее задание

1. Создайте условие, которое проверяет, является ли число четным или нечетным, и выводит соответствующее сообщение.

let число = 10;
if (число % 2 === 0) {
    console.log('Число четное.');
} else {
    console.log('Число нечетное.');
}

2. Напишите программу, которая использует конструкцию switch для вывода названия дня недели по номеру (1 — понедельник, 2 — вторник и т.д.).

let день = 3;
switch (день) {
    case 1:
        console.log('Понедельник');
        break;
    case 2:
        console.log('Вторник');
        break;
    case 3:
        console.log('Среда');
        break;
    default:
        console.log('Некорректный номер');
}

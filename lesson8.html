Урок 8: Циклы

Введение

Преподаватель: На прошлом уроке мы изучали условия и логические операторы. Сегодня мы поговорим о циклах. Циклы позволяют выполнять блок кода несколько раз, что делает ваши программы более эффективными и сокращает количество дублирующегося кода.

В конце урока вы сможете использовать циклы for, while и do...while.

Теоретическая часть

Преподаватель: Циклы в JavaScript помогают выполнять повторяющиеся задачи. Основные виды циклов:

  • for: используется, когда заранее известно количество итераций.
  • while: продолжает выполнение, пока условие истинно.
  • do...while: выполняет блок кода хотя бы один раз, даже если условие ложно.

Интересный момент: в JavaScript циклы можно использовать вместе с массивами для перебора их элементов. Это похоже на использование циклов в C++ и Python, но с небольшими отличиями в синтаксисе. Например, в JavaScript у вас есть метод forEach для массивов, который упрощает перебор элементов.

Пример кода

Преподаватель: Давайте рассмотрим простой пример использования циклов:

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Циклы в JavaScript</title>
</head>
<body>
    <script>
        // Цикл for
        for (let i = 0; i < 5; i++) {
            console.log('Итерация ' + i);
        }

        // Цикл while
        let j = 0;
        while (j < 5) {
            console.log('Итерация ' + j);
            j++;
        }

        // Цикл do...while
        let k = 0;
        do {
            console.log('Итерация ' + k);
            k++;
        } while (k < 5);
    </script>
</body>
</html>

В этом коде мы используем три разных цикла для выполнения одной и той же задачи — выводим итерации от 0 до 4.

for (let i = 0; i <= 20; i++) {
    if (i % 2 === 0) {
        console.log(i);
    }

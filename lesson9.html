Урок 9: Функции

Введение

Преподаватель: На прошлом уроке мы изучали циклы. Сегодня мы поговорим о функциях. Функции позволяют нам организовывать код и повторно использовать его, что делает программы более структурированными и легкими для понимания.

В конце урока вы сможете создавать функции, передавать в них параметры и возвращать значения.

Теоретическая часть

Преподаватель: Функции в JavaScript — это блоки кода, которые можно вызывать по имени. Основные моменты:

  • Объявление функции с помощью ключевого слова function.
  • Передача параметров в функцию для работы с различными данными.
  • Возврат значений с помощью ключевого слова return.

Интересный момент: в JavaScript функции являются объектами первого класса, что означает, что их можно передавать как аргументы другим функциям, возвращать из функций и сохранять в переменных. Это отличие от C++ и Python, где функции также могут быть объектами, но синтаксис и поведение немного отличаются.

Пример кода

Преподаватель: Давайте рассмотрим простой пример функции:

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Функции в JavaScript</title>
</head>
<body>
    <script>
        // Объявление функции
        function сложить(a, b) {
            return a + b;
        }

        // Вызов функции
        console.log(сложить(3, 5)); // 8
    </script>
</body>
</html>

В этом коде мы создаем функцию сложить, которая принимает два параметра и возвращает их сумму. Мы затем вызываем эту функцию и выводим результат в консоль.

Домашнее задание

1. Напишите функцию, которая принимает массив чисел и возвращает их среднее значение.

function среднее(arr) {
    let сумма = 0;
    for (let i = 0; i < arr.length; i++) {
        сумма += arr[i];
    }
    return сумма / arr.length;
}

2. Создайте функцию, которая принимает два числа и возвращает большее из них.

function большее(a, b) {
    return a > b ? a : b;
}

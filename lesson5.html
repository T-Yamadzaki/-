Урок 5: Функции

Введение

Преподаватель: На прошлом уроке мы изучали циклы. Сегодня мы поговорим о функциях. Функции позволяют организовать код в удобные и переиспользуемые блоки, что улучшает его структуру и читаемость.

В конце урока вы сможете создавать функции и использовать параметры и возвращаемые значения.

Теоретическая часть

Преподаватель: Функции в JavaScript — это набор инструкций, которые выполняются при их вызове. Основные моменты:

  • Функции могут принимать параметры и возвращать значения.
  • Функции могут быть объявлены с помощью ключевого слова function.
  • В JavaScript функции являются объектами и могут быть присвоены переменным, переданы как аргументы и возвращены из других функций.

В отличие от C++, в JavaScript вы не обязаны объявлять типы возвращаемых значений, так как язык динамически типизирован. В Python функции также могут возвращать значения, но синтаксис отличается.

Пример кода

Преподаватель: Давайте рассмотрим простой пример функции, которая принимает два числа и возвращает их сумму:

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Функции в JavaScript</title>
</head>
<body>
    <script>
        function сложить(a, b) {
            return a + b;
        }

        console.log(сложить(3, 5)); // Выводит 8
    </script>
</body>
</html>

В этом коде мы определяем функцию сложить, которая принимает два параметра и возвращает их сумму.

Домашнее задание

1. Создайте функцию, которая принимает массив чисел и возвращает их среднее значение.

function среднее(числа) {
    let сумма = 0;
    for (let i = 0; i < числа.length; i++) {
        сумма += числа[i];
    }
    return сумма / числа.length;
}

2. Напишите функцию, которая принимает строку и возвращает её в обратном порядке.

function обратнаяСтрока(str) {
    return str.split('').reverse().join('');
}
